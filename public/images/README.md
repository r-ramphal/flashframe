# Afbeeldingen

Plaats hier je eigen foto's om de placeholders te vervangen:

- `spinnerbooth.jpg` — foto van de 360° spinnerbooth
- `fotobooth.jpg` — foto van de fotobooth met print

Aanbevolen formaat: liggend (4:3 verhouding), minimaal 1200×900 px.
Zolang een bestand ontbreekt, toont de site automatisch een nette placeholder.
